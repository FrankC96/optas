# OPTAS

`OPTAS` is an OPtimization-based TAsk Specification library for trajectory optimization and model predictive control. 

# Install

## From source

1. `$ git clone git@github.com:cmower/optas.git`
2. `$ cd optas`
3. `$ pip install .`
